    FAQ

  • General

  • 센드애니웨어는 어떤 서비스인가요?

    6자리 키를 이용해 다양한 디바이스 사이에도 쉽고 간편하게 파일을 전송할 수 있는 서비스입니다.

  • 양쪽 기기 모두에 센드애니웨어를 설치해야 하나요?

    스마트폰 등의 기기로 파일을 주고 받을 경우 파일을 보내고 받는 기기 양쪽 모두 센드애니웨어가 설치되어 있어야 합니다. PC의 경우 설치 없이 센드애니웨어 홈페이지 https://send-anywhere.com/에서도 사용이 가능합니다.

  • 파일은 서버에 얼마동안 보관되나요?

    보통 전송의 경우 10분 동안, 24시간 파일 전송의 경우 24시간 동안만 서버에 보관됩니다.

  • 10분 전송과 24시간 전송의 차이가 뭔가요?

    10분 전송은 10분 간 유효하며 단 한번만 사용이 가능합니다. 그렇기 때문에 1:1 전송에 사용됩니다. 24시간 전송은 24시간 동안 유효하며 6자리 키를 알고 있다면 여러 번 파일을 다운 받을 수 있습니다. 주로 1: 다수 전송에 사용됩니다.

  • 보내고 삭제한 파일을 복구할 수 있나요?

    센드애니웨어 앱 내의 히스토리에서 삭제한 경우라면 파일 전송 기록만 삭제될 뿐, 실제로 파일이 삭제되지는 않습니다.그러나 기기/PC에서 직접 파일을 삭제한 경우에는 삭제한 파일을 복구할 수 없습니다.

  • 6자리 키가 뭔가요?

    파일 보내기를 완료하면 만들어지는 키입니다. 이 6자리 키를 파일을 받고자 하는 기기의 '받기'에 입력하면 해당 파일을 다운받을 수 있습니다.

  • 파일은 어떻게 보내는 건가요?

    10분 전송은 10분 간 유효하며 단 한번만 사용이 가능합니다. 그렇기 때문에 1:1 전송에 사용됩니다. 24시간 전송은 24시간 동안 유효하며 6자리 키를 알고 있다면 여러 번 파일을 다운 받을 수 있습니다. 주로 1: 다수 전송에 사용됩니다.

  • 10분 전송과 24시간 전송의 차이가 뭔가요?

    1. 센드애니웨어 앱/홈페이지에 들어가서 보낼 파일을 선택합니다.
    2. '키 생성'을 눌러줍니다.
    3. 만들어진 6자리 키를 파일을 다운받을 기기의 센드애니웨어 앱/홈페이지의 '받기'에 입력하면 파일이 전송됩니다.

  • 어떤 기기에서 사용이 가능한가요?

    현재 안드로이드, ios, 윈도우폰, 윈도우 데스크탑, 맥, 리눅스, 크롬에서 사용이 가능합니다.

  • 10분 전송과 24시간 전송의 차이가 뭔가요?

    10분 전송은 10분 간 유효하며 단 한번만 사용이 가능합니다. 그렇기 때문에 1:1 전송에 사용됩니다. 24시간 전송은 24시간 동안 유효하며 6자리 키를 알고 있다면 여러 번 파일을 다운 받을 수 있습니다. 주로 1: 다수 전송에 사용됩니다.

  • 파일의 용량이나 개수, 종류에 제한이 있나요?

    용량이나 개수, 종류에 제한이 없습니다.

  • 회원가입을 해야만 파일을 주고받을 수 있나요?

    회원 가입을 하지 않아도 6자리 키만 알면 파일을 주고받을 수 있습니다.

  • 내 디바이스는 어떤 기능인가요?

    내가 가지고 있는 여러 기기에 동일한 아이디로 로그인을 하면 일일이 6자리 키를 입력 할 필요 없이 클릭 한 번으로 내 기기 사이에 쉽게 파일을 주고 받을 수 있습니다.

  • 로그인은 어떻게 하는 건가요?

    센드애니웨어 메인에서 로그인을 누르고, 회원가입한 아이디 메일과 비밀번호를 입력하면 됩니다.

  • 모르는 사람들이 핸드폰에 표시되는데 왜 그런거죠? 내 디바이스가 보이지 않게 할 수는 없나요?

    근처에 있는 스마트 기기들 중에서 센드애니웨어가 설치되어 있는 기기 리스트를 보여 주는 것으로 연결이 되어 있는 것은 아닙니다. 다른 사용자의 기기에 내 기기가 표시되지 않도록 하고 싶다면 메뉴>앱 설정을 클릭하고 '검색 가능'의 체크 표시를 해제하시면 됩니다.

  • 전송 대기중일 때 디바이스에서 Recently는 어떤건가요?

    최근에 파일을 주고 받은 기록이 있는 기기를 Recently로 보여 줍니다.

  • 센드애니웨어로 파일을 주고받는 것은 안전한가요?

    센드애니웨어로 파일을 전송할 경우 단 10분만 6자리 키가 유효하며 한 번 사용된 키는 그 이상 사용이 불가능합니다.그렇기 때문에 10분 안에 6자리 키를 해킹하는 것은 매우 어려우며, 시간이 지난 파일은 서버에서 삭제되기 때문에 서버 해킹을 당한다고 해도 파일이 해킹당할 염려가 없습니다.24시간 전송의 경우 숫자와 알파벳이 결합된 보다 보안이 강화된 6자리 키를 사용하고 있으며 역시 24시간이 지나면 서버에서 파일이 삭제됩니다.

  • 누군지 모르는 상대가 파일을 보냈다는 메세지가 왔어요. 어떻게 해야 하나요?

    보통의 경우 Nearby디바이스를 잘못 클릭해 파일을 보내는 경우입니다. 모르는 디바이스라면 파일 전송을 받지 않으시는 것이 보다 안전합니다.

  • 숫자키는 여러 번 사용이 가능한가요?

    숫자키는 단 한번만 사용 가능하며 10분 동안만 유효합니다.

  • 누군지 모르는 상대가 파일을 보냈다는 메세지가 왔어요. 어떻게 해야 하나요?

    보통의 경우 Nearby디바이스를 잘못 클릭해 파일을 보내는 경우입니다. 모르는 디바이스라면 파일 전송을 받지 않으시는 것이 보다 안전합니다.

  • 숫자키는 여러 번 사용이 가능한가요?

    숫자키는 단 한번만 사용 가능하며 10분 동안만 유효합니다.

  • 받은 파일은 어디에 저장되나요?

    보통의 경우 사진이나 영상은 앨범에 저장되며 그 외 파일은 센드애니웨어 폴더에 저장됩니다. 저장 폴더는 메뉴>앱 설정에서 '받은 파일 경로'를 보면 확인이 가능합니다.

  • 파일을 저장하는 위치를 변경하고 싶어요. 어떻게 하나요?

    저장 폴더는 메뉴>앱 설정에서 '받은 파일 경로'를 클릭하면 원하는 위치로 변경이 가능합니다.

  • 센드애니웨어를 이용하려면 인터넷 연결이 필수인가요?

    센드애니웨어는 P2P와 유사한 시스템을 기반으로 하기 때문에 꼭 인터넷 연결이 가능해야 합니다.

  • 6자리 숫자키를 친구에게 알려 주고 싶어요. 어떻게 하면 될까요?

    -PC버전의 경우 Click above key to copy를 눌러서 복사된 키 번호를 원하는 SNS로 공유하면 편하게 키번호 공유가 가능합니다.

    -스마트폰의 경우 6자리 키의 하단에 보면 '공유' 탭이 있습니다. 여기를 누르면 원하는 SNS나 메일, 문자 등의 방법으로 쉽게 키번호 공유가 가능합니다.

  • 보낼 파일을 잘못 선택했어요. 전송 중단이 가능한가요?

    전송 중에 취소 또는 cancel 버튼을 누르면 전송이 중단됩니다.

  • 주고받은 파일 내역은 어디서 확인이 가능한가요?

    메뉴의 히스토리에서 이제까지 주고 받은 파일 전송 목록을 확인 할 수 있습니다.

  • 비밀번호를 잊어버렸어요. 어떻게 하나요?

    -데스크탑 버전의 경우 로그인 화면에서 Forgot password를 클릭하고 이메일 주소를 입력하면 비밀번호를 변경할 수 있습니다.

    -스마트폰의 경우 로그인 화면 하단에 '암호를 잊으셨습니까?'를 클릭하고 이메일 주소를 입력하면 비밀번호를 변경할 수 있습니다.

  • 비밀번호를 변경하고 싶어요.

    -데스크탑 버전의 경우 로그인 화면에서 Forgot password를 클릭하면 이메일 주소를 확인한 후 비밀번호를 변경하는 화면이 나옵니다.

    -스마트폰의 경우 로그인한 상태에서 메뉴>앱 설정으로 들어가면 '패스워드 변경' 이 있습니다. 이곳을 클릭해 원하는 비밀번호를 재설정하면 됩니다.

  • 디바이스 이름을 바꾸고 싶어요. 어떻게 하나요?

    -데스크탑 버전의 경우 왼쪽 상단의 연필 아이콘을 누르면 원하는 이름으로 디바이스 명 변경이 가능합니다.

    -스마트폰의 경우 메뉴>앱 설정으로 들어가면 '디바이스 명 변경'이 있습니다. 이 곳을 클릭해 원하는 디바이스 명을 설정하면 됩니다.

  • 파일 삭제는 어떻게 하나요?

    센드애니웨어 앱에서는 파일을 바로 삭제할 수가 없습니다. 기기/PC에서 직접 삭제하셔야만 정상적으로 파일이 삭제됩니다.

  • 양쪽이 같은 네트워크에 있어야 파일 전송이 가능한가요?

    같은 네트워크에 있을 필요는 없습니다. 서로 다른 네트워크에 있더라도 인터넷 연결이 되어 있으면 파일 전송이 가능합니다.

  • 파일을 여러명이 동시에 받으면 데이터 사용량이 커지나요?

    24시간 업로드를 통해서 여러 사람이 다운받을 경우에 파일 보내는 데이터 소모는 1번 업로드 하는 것만 고려하시면 됩니다.

  • 스마트폰으로 파일을 전송할 때는 빠른데 PC로 보내면 느려요.

    센드애니웨어의 속도는 기본적으로 인터넷 속도에 따라 좌우됩니다. 그 중에서도 웹이나 크롬의 경우 서버릴레이 전송을 하기 때문에 더 느리다고 느껴질 수가 있는데 이 경우에는 P2P 연결을 기반으로 하는 윈도우 데스크탑 버전 등을 설치해 사용하면 좀 더 빠르게 파일 전송을 할 수 있습니다.

  • 센드애니웨어 홈페이지에서 자꾸 오류가 나요. 왜 그런거죠?

    현재 센드애니웨어는 windows IE10 이상에서 정상적으로 사용이 가능합니다. 이 경우 익스플로어 버전을 업데이트 하시거나 크롬과 맥 같은 다른 PC버전을 사용할 경우 정상적으로 파일을 전송할 수 있습니다.

  • 모든 PC버전에서 다 로그인이 가능한가요?

    그렇지 않습니다. 현재 PC버전을 로그인해서 사용하고 싶다면 윈도우 데스크탑 버전을 사용하셔야만 로그인 사용이 가능합니다.

  • Nearby Devices가 뭔가요?

    동일 네트워크 상에 있는 기기들 중 센드애니웨어가 설치되어 있는 기기들을 보여줍니다. 자신의 기기가 타인의 Nearby Devices 리스트에 나오는 않도록 하고 싶다면 Menu>setting>app setting에서 '검색 가능' 의 체크를 해제하면 됩니다.

  • 전송 대기중일 때 디바이스에서 Nearby는 어떤건가요?

    같은 네트워크 상에 있는 스마트 기기들 중에서 센드애니웨어가 설치되어 있는 기기를 보여 줍니다. 클릭할 경우 6자리 키를 입력 할 필요 없이 원클릭으로 쉽게 파일을 전송할 수 있습니다.

  • 파일을 주고받는 동안 데이터 요금이 드나요?

    센드애니웨어는 인터넷 연결을 기반으로 하기 때문에 파일을 주고 받는 동안 인터넷 연결을 한 것과 동일한 데이터 요금이 나올 수 있습니다.그렇기 때문에 주고받고자 하는 파일의 크기가 큰 경우 가급적 Wifi 연결을 하는 것을 권장합니다.

  • 파일을 주고받는데 와이파이로 받는지, 셀룰러 데이터로 받는지 어떻게 알?

    센드애니웨어의 화면 오른쪽 상단에 노란색 글씨로 현재 사용하는 데이터에 관해 표시해 드리고 있습니다.

  • 동영상을 받았는데 재생이 안되요. 어떻게 하나요?

    이 경우 해당 동영상의 코덱을 우선적으로 확인해야 합니다.해당 동영상의 코덱이 플레이어가 지원하지 않는 코덱인 경우 지원하는 동영상 플레이어를 다운받으면 정상적으로 동영상 재생이 가능합니다.

  • 앱을 받았는데 설치가 안되요. 어떻게 하나요?

    같은 앱이더라도 단말기에 맞춰서 앱을 설정을 별개로 하고 배포하는 경우가 있기 때문에 다운로드한 앱이 설치하고자 하는 단말기에 설치될 수 있도록 설정이 되어 있지 않으면 앱이 정상적으로 설치 되지 않습니다.

  • 파일을 보내기 전에 제대로 파일을 선택했는지 알고 싶어요. 어디서 보면 되나요?

    파일을 선택하는 화면 하단에 파일의 개수가 표시됩니다. 개수가 표시된 검은색 탭을 누르면 선택한 파일의 목록을 다시 한 번 확인 할 수 있습니다.

  • 파일을 선택했는데 그 중에 몇 개는 빼고 보내고 싶어요. 가능한가요?

    파일을 선택하는 화면 하단에 보내는 파일의 개수가 표시된 검은색 탭을 누르면 선택한 파일의 목록이 나옵니다. 여기서 보내고 싶지 않은 파일을 선택하면 하단 파일 개수가 표시된 곳에 휴지통 모양의 아이콘이 생깁니다. 이 휴지통 모양의 아이콘을 누르면 해당 파일이 보내는 파일 목록에서 사라집니다.

  • 사진과 동영상을 함께 보낼 수 있나요?

    사진을 선택 한 후 바로 '키 생성' 버튼을 누르지 않고 원하는 파일 포맷이 나올 때까지 옆으로 터치합니다. 변경된 파일 포맷에서 원하는 파일을 선택하면 하단에 파일 개수가 변경된 것을 볼 수 있습니다. 원하는 모든 파일을 선택한 다음 '키 생성' 버튼을 누르면 하나의 키번호로 여러 포맷의 원하는 파일들을 전송할 수 있습니다.

  • 숫자키 자리에 있는 '사용법'은 무엇인가요?

    처음 사용하는 사용자를 위해 튜토리얼을 다운할 수 있도록 한 것입니다. 다운 방법은 받기 하단에 있는 글 중 원하는 글의 파란색 표시된 단어를 선택하고 다운 버튼을 누르면 됩니다.

  • 안드로이드에서 아이폰으로 파일을 보낼 수도 있나요?

    저희가 지원하는 플랫폼이라면 기기에 상관없이 간편하게 파일을 보낼 수 있습니다.현재 지원하는 플랫폼은 안드로이드, iOS, 윈도우폰, windows 8, Mac, Chrome, Linux, Window desktop 버전을 제공하고 있습니다.

     

  • iOS

  • 음악 파일을 센드애니웨어로 다운받았는데 왜 뮤직 플레이어에서는 보이지 않죠?

    음악 파일의 경우 ios기기 자체의 뮤직 앱으로는 재생이 불가능하답니다. 이것은 iOS에서 iTunes외의 3rd party가 음악 파일을 library에 저장하는 것을 금지했기 때문인데요, 그래서 음악 파일 하나씩을 센드애니웨어 앱 내의 "My Documents" 에서 듣는 것만 가능합니다.

  • 왜 음악 파일을 선택해서 전송할 수 없는 거죠?

    iOS에서 3rd party 앱이 음악 파일을 library에서 다른 기기로 이동시키는 것을 금지했기 때문에 전송이 불가능합니다.

  • 내 아이폰/아이패드로 보낸 파일을 못찾겠어요.

    이 경우 우선 센드애니웨어 앱의 "My Documents" 를 확인해 보시기 바랍니다. 만일 "My Documents"에서도 찾을 수 없는 경우 아이폰/아이패드를 PC의 iTunes에 연결해 “iTunes>Device>App>File Sharing>Send Anywhere" 에서 찾아보시기 바랍니다. 만일 여기서도 찾을 수 없는 것은 iOS에서 해당 파일의 포맷을 지원하지 않는 경우입니다.

  • 왜 내 앱을 전송할 수 없는 건가요?

    iOS의 경우 바로 앱을 전송받거나 전송할 수 없습니다. 앱 전송의 경우 현재는 안드로이드 기기 사이에서만 가능합니다.

  • 다른 기기에서 동영상을 전송 받았는데 재생 할 수가 없어요.

    받은 동영상 파일의 포맷을 iOS가 지원하지 않는 경우입니다. iOS는 표준화 및 인증된 몇 종류의 비디오 포맷 및 코덱만을 지원합니다.

    *iOS 지원 코덱

    -H.264 video, up to 1.5 Mbps, 640 by 480 pixels, 30 frames per second, Low-Complexity version of the H.264 Baseline Profile with AAC-LC audio up to 160 Kbps, 48 kHz, stereo audio in .m4v, .mp4, and .mov file formats

    -H.264 video, up to 768 Kbps, 320 by 240 pixels, 30 frames per second, Baseline Profile up to Level 1.3 with AAC-LC audio up to 160 Kbps, 48 kHz, stereo audio in .m4v, .mp4, and .mov file formats

    -MPEG-4 video, up to 2.5 Mbps, 640 by 480 pixels, 30 frames per second, Simple Profile with AAC-LC audio up to 160 Kbps, 48 kHz, stereo audio in .m4v, .mp4, and .mov file formats

    만일 iOS가 지원하지 않는 동영상의 경우라면 해당 포맷/코덱을 지원하는 동영상 플레이어 앱을 다운받아야 정상적으로 동영상을 재생할 수 있습니다..

  • 보이스 녹음 파일은 어떻게 전송해야 하나요?

    1. 우선 보이스 메모 앱을 열고 원하는 녹음 파일을 선택하세요.

    2. 원하는 파일을 탭 해서 쉐어 버튼을 누르면 메일이나 메세지 등을 통해 공유할 수 있게 됩니다. 여기서 "More" 을 눌러서 리스트에서 센드애니웨어를 선택해 줍니다.(이 과정을 위해서는 먼저 센드애니웨어 앱이 설치되어 있어야 합니다.)

    3. 센드애니웨어가 쉐어가 가능하도록 스위치를 활성화시켜 주면 됩니다. 그 다음부터는 원하는 녹음파일을 선택, 쉐어 버튼을 눌러주기만 하면 쉽게 녹음 파일 전송이 가능합니다.

    FAQ

  • General

  • How does Send Anywhere work?

    Send Anywhere pairs two devices with a temporarily generated key confirmation and transfers files instantly via the optimal network path between the devices.

  • How can I send/receive files?
    By a Temporarily Generated 6-Digit Key

    1.Select files to send and hit the “Send” button on your device.

    2.Share the temporarily generated 6-digit key with the receiving device.

    3.The transfer process begins when the receiving device inputs the key.

    By Selecting a Receiving Device

    1.Select files to send and hit the “Send” button on your device.

    2.Select the device that you want to send the files to from the list of nearby or recently used devices.

    3.Transferring begins instantly when the receiving device confirms the transfer.

  • Is Send Anywhere secure?

    We use SSL encryption on our website https://send-anywhere.com and mobile apps; however, keeping the temporarily generated 6-digit key away from others is important. You can get enhanced security keys – numerical and alphabetical mixtures – through the application’s settings. Sending files through Send Anywhere is completely secure because the files are never stored on any servers. Instead, they are transferred directly via a peer-to-peer network connection between devices. If the devices are in remote locations or in different networks, the p2p path may sometimes fail. In these instances, Send Anywhere finds the most effective node-server to pass through without storing any files. The process for uploading files for later download follows the same convention as a normal upload/download process. The maximum allotted file storage time is 24 hours so there is no need to worry about security.

  • Do I need a network connection for Send Anywhere to work?

    Yes, you need an Internet connection so that the Send Anywhere app can communicate with our servers. Send Anywhere uses whatever Internet connection is available on your device—whether it be WiFi or cellular.

  • Does Send Anywhere use my phone’s data plan? Will I be charged?

    Yes, if you are connected to the Internet via a cellular service, Send Anywhere will use your data plan. Your standard data plan rates will apply.

  • Does Send Anywhere require Bluetooth activation?

    Nope! Send Anywhere does not use Bluetooth; all you need is an Internet connection.

  • Is there a file size limit when not uploading to the server?

    There is no limitation on file size but storage expiration time for uploaded files is 24 hours.

  • I just downloaded files from the website, where did they go?

    If you used our website to complete the file transfer process, please check the “Downloads” folder on your computer to locate the downloaded files.

  • Does Send Anywhere transfer files using a p2p connection?
    When sending files directly, the transfer path depends on the network connection between two devices.

    If both two devices are in the same local network, it’s 100% p2p.

    If either device is using 3G, 4G, or 4G LTE then it depends on distance, carrier network’s base stations, etc.

     

  • iOS

  • Why can’t I see the received music files in my music player?

    Received music files won’t be available on your native music application. iOS does not allow 3rd party apps to save music files in your music library—you can only do this via an iTunes connection. You can play a single music file received on your Send Anywhere app by previewing it in your “My Documents” tab on the bottom menu.

  • I can’t find the files that I sent to my iPhone/iPad

    If this is the case, the first thing that you should do is check in your “My Documents” tab on the bottom menu. If you cannot locate it here, please connect your iOS device to your PC via iTunes and look into “iTunes>Device>App>File Sharing>Send Anywhere”. If you still cannot locate it, there is the possibility that iOS does not support the file format that you sent to it.

  • Why can’t I transfer my apps?

    On iOS, apps are not allowed to be transferred directly from/to other device. Transferring apps with Send Anywhere only stands for Android devices.

  • I cannot play the video file that I’ve received from other device

    The video file you received might not be a supported format on iOS.iOS supports many industry-standard video formats and compression standards, including the following:

    1. H.264 video, up to 1.5 Mbps, 640 by 480 pixels, 30 frames per second, Low-Complexity version of the H.264 Baseline Profile with AAC-LC audio up to 160 Kbps, 48 kHz, stereo audio in .m4v, .mp4, and .mov file formats

    2. H.264 video, up to 768 Kbps, 320 by 240 pixels, 30 frames per second, Baseline Profile up to Level 1.3 with AAC-LC audio up to 160 Kbps, 48 kHz, stereo audio in .m4v, .mp4, and .mov file formats

    3. MPEG-4 video, up to 2.5 Mbps, 640 by 480 pixels, 30 frames per second, Simple Profile with AAC-LC audio up to 160 Kbps, 48 kHz, stereo audio in .m4v, .mp4, and .mov file formats

    If it’s not natively supported by iOS, you’ll need to open it with a third-party app that supports the format.

  • How can I transfer Voice memo recording file?

    On your Voice Memos app, select a recording file and tap share button. By default you can share the file using Message and Mail. Now Tap “More” button and you will see “Send Anywhere” on the list of all possible apps sharing activities(Ensure that you have Send Anywhere app installed in your phone). By turning on the switch of Send Anywhere, than you can share your recording files using Send Anywhere.
